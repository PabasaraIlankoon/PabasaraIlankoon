### 🐘 Elevision | AI-Powered Elephant Detection & Early Warning System

**Status:** 🔥 Preprint manuscript under preparation &nbsp;|&nbsp; **Domain:** Railway Safety × Edge AI

Real-time computer-vision system that detects elephants near Sri Lankan railway tracks and instantly alerts
operators, cross-referencing live train schedules to flag exactly which trains are at risk.

- 🧠 YOLOv8-based detection models trained for elephant recognition in railway environments
- ⚡ Edge AI inference on Raspberry Pi via **ONNX**, optimised for real-time performance
- ☁️ Firebase, GSM fallback, and Flutter integration for remote monitoring and instant alerts
- 🌐 Companion **React.js web dashboard** for control-room visualisation

### 🤖 Robotics & Embedded Systems

<details open>
<summary><b>🔥 Mars Robot Competition | Autonomous Robotics System</b></summary>
<br>

Autonomous robot for competition navigation and object handling. Sensor fusion (IR, ultrasonic, gyroscope, color
sensors), real-time decision-making on ESP32-S3, SolidWorks + 3D-printed chassis, encoder-based precision motor
control via TB6612FNG.

`ESP32-S3` `C/C++` `MPU6050` `TCS34725` `SolidWorks` `Embedded Systems`
</details>

<details>
<summary><b>ROSCO'25 Competition Robot | Autonomous Line & Wall Following Robot</b></summary>
<br>

Autonomous line following, wall following, and ramp navigation using VL53L0X ToF sensors, IR array, and MPU6050
IMU sensor fusion. Laser-cut acrylic chassis with custom 3D-printed parts and encoder feedback for motion stability.

`ESP32-S3` `VL53L0X` `IR Sensors` `MPU6050` `TB6612FNG` `SolidWorks`
</details>

<details>
<summary><b>LankaMesh | LoRa-Based Disaster Communication System</b></summary>
<br>

Decentralized emergency communication system for disaster zones with no internet or cellular coverage. Long-range
peer-to-peer LoRa communication for SOS alerts, messages, and GPS sharing, with a Flutter app for emergency alert
categorisation.

`ESP32-S3` `LoRa RA-02 (SX1278)` `GPS` `Flutter` `IoT` `Wireless Communication`
</details>

### 🧠 AI, ML & Computer Vision

<details open>
<summary><b>🔥 AgroVision AI | Intelligent Tomato Leaf Disease Detection System</b></summary>
<br>

Full-stack AI system for plant disease detection using a MobileNetV2 transfer-learning model for multi-class
classification, with React.js + Flutter clients, a FastAPI backend, and Firebase for image storage and prediction
history.

`TensorFlow` `Keras` `OpenCV` `FastAPI` `React.js` `Flutter` `Firebase` `MobileNetV2`
</details>

<details>
<summary><b>Digital Communication Simulator</b></summary>
<br>

Web-based simulation platform modelling digital communication systems - ASK, QPSK, 16-QAM, and OFDM modulation,
AWGN channel effects, and BER vs SNR performance evaluation with interactive Plotly/Matplotlib visualisations.

`Python` `Flask` `NumPy` `SciPy` `Plotly` `Matplotlib` `DSP`
</details>
